ОТЧЕТ ПО ПРАКТИЧЕСКОЙ РАБОТЕ
Классификация текстов на естественном языке с помощью машинного обучения

================================================================================

1. ПОСТАНОВКА ЗАДАЧИ

Цель работы: Разработать программу для классификации текстов на русском языке
с использованием различных методов векторизации и алгоритмов машинного обучения.

Задачи:
- Реализовать лемматизацию текстов
- Реализовать частотную векторизацию (TF-IDF)
- Реализовать семантическую векторизацию (Word2Vec)
- Обучить модели классификации (логистическая регрессия, случайный лес, 
  наивный байесовский классификатор)
- Сравнить результаты различных подходов

================================================================================

2. ИСПОЛЬЗУЕМЫЕ ИНСТРУМЕНТЫ

- Python 3.13.1
- Библиотеки: NumPy, Pandas, scikit-learn, pymorphy3, Gensim, Matplotlib
- Датасет: 300 примеров (синтетический датасет для классификации тональности)

================================================================================

3. МЕТОДЫ

3.1. Предобработка текстов:
    - Очистка от специальных символов
    - Лемматизация с помощью pymorphy3

3.2. Векторизация:
    - TF-IDF: частотный подход, 251 признак
    - Word2Vec: семантический подход, 100 признаков

3.3. Алгоритмы классификации:
    - Логистическая регрессия
    - Случайный лес (100 деревьев)
    - Наивный байесовский классификатор

================================================================================

4. РЕЗУЛЬТАТЫ

Точность моделей на тестовой выборке (60 примеров):

┌──────────────────────────────────┬──────────┬──────────┐
│ Алгоритм                         │ TF-IDF   │ Word2Vec │
├──────────────────────────────────┼──────────┼──────────┤
│ Логистическая регрессия          │ 1.0000   │ 0.8500   │
│ Случайный лес                    │ 1.0000   │ 1.0000   │
│ Наивный байесовский классификатор│ 1.0000   │ 0.8500   │
└──────────────────────────────────┴──────────┴──────────┘

Выводы:
1. TF-IDF показал превосходные результаты (100% точность)
2. Word2Vec показал хорошие результаты (85-100% в зависимости от алгоритма)
3. Случайный лес показал одинаково высокие результаты с обоими подходами
4. Лемматизация успешно применена ко всем текстам

================================================================================

5. ЗАКЛЮЧЕНИЕ

Все поставленные задачи выполнены успешно. Разработана полнофункциональная
система классификации текстов на русском языке с возможностью сравнения
различных подходов к векторизации и алгоритмов классификации.

Полученные результаты демонстрируют эффективность частотного подхода (TF-IDF)
для данной задачи, при этом семантический подход (Word2Vec) также показал
хорошие результаты, особенно в сочетании со случайным лесом.

Система может быть использована для решения практических задач анализа
текстов: классификация отзывов, категоризация новостей, определение тональности.
